## ✨ Key Features

### 🔐 Authentication System
- Email & Password Registration and Login
- Google OAuth Social Login
- Protected Routes for authenticated users only
- User session management with BetterAuth

### 🛍️ Product System
- Browse all summer products on Products page
- View detailed product information (protected)
- Product cards with image, name, rating, price
- Add to Cart & Wishlist buttons

### 👤 User Profile
- View profile with avatar, name, and email
- Update profile name and photo URL
- Responsive profile page

### 🎨 UI/UX
- Fully responsive (mobile, tablet, desktop)
- Dark theme with pink accent colors
- Smooth hover animations and transitions
- Sticky navbar with dynamic auth state
